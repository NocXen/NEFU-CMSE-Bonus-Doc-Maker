# 介绍

这是一个特定场景项目：

适用于：东北林业大学材料科学与工程学院的加分审批单及回单的制作。

# 怎么用？

1. `Code - Download ZIP`，解压到任意文件夹。

2. 安装python，添加到PATH。右键win打开powershell，执行：

```bash
pip install pandas openpyxl python-docx docxcompose
```

该步骤只用执行一次。

3. 清空`./Output_输出/`。如果已经为空，跳过。

4. 在`./Input_输入/`放入`.xlsx`格式的汇总表。表头必须有：`班级`, `姓名`, `事由`, `性质`, `分数`, `日期`。

5. 执行`./Filler.py`。若成功，会显示`按回车键退出`。文件会自动生成在`./Output_输出/`下。

# 其他？

1. `./Checker.py`

检查`./Input_输入/`中的表格是否有重复行。会输出一个日志。

2. `./班级标准转换/名称转换251111.py`

输入含有任何奇形怪状的班级名的表格 ->

启动！ ->

输出含有整齐正式的班级名的表格。
