### Python 文档替换工具使用说明书
## 一、工具概述
# 本工具用于自动化生成学院审批单和回执单文档。
# 通过填写简单的数据文件，
# 即可批量替换 Word 文档中的占位符字段，
# 快速生成规范的学院文件。

## 二、数据文件格式说明
# 数据文件（如 `文档替换_data.txt`）
# 采用结构化格式编写：
#   注释行：以 `` 或 `--` 开头，用于说明和分隔
#   字段行：以 `-  $ ` 开头，
#   格式：`-  $ 字段名 注释说明:`
#   值行：紧跟字段行下方，填写具体数值。
#     注：值可以是多行。
#     以 `Output.FileName` 的行数为基准行数，
#     支持批量多次生成。
#     若其他字段行数小于基准行数，
#     将使用最后一行数据进行生成。
#
# 示例：
#   -  $ Reason.Name 人名 :
#   张三 李四 王五

## 三、特殊字段说明
# 1. 拆分字段：
#   字段名以 `.split(分隔符)` 结尾（如 `RN.split(，)`）
#   工具会自动拆分值并生成多个字段。
#   （如 `RN.split.1`, `RN.split.2` 等）
   
# 2. 自动转换字段：
#   以 `.CL.int` 结尾:
#       转换为正式大写数字（壹、贰）
#   以 `.CLS.int` 结尾:
#       转换为非正式大写数字（一、二）
#   日期字段:
#       自动格式化为 `YYYY.MM.DD` 或
#        `YYYY年MM月DD日`

## 四、使用流程
#   步骤1：准备模板文件
#   创建包含占位符字段的 Word 文档。

#   步骤2：填写数据文件
#   1. 复制 `DR_data.txt` 文件
#   2. 按以下顺序填写：
#     -  $ Template.FileName 模板文件:
#     模板1.docx 模板2.docx
#     -  $ Output.FileName 输出文件:
#     输出1.docx 输出2.docx
#     -  $ Reason.Name 人名:
#     张三 李四 王五
#     -  $ Score.type 加分性质:
#     德育
#     ...（其他字段）

#   步骤3：运行脚本

#   步骤4：获取结果
#   在脚本指定目录下生成：
#   `输出1.docx`、`输出2.docx`


## 五、注意事项
# 1. 字段顺序：必须保持文件中字段的顺序
# 2. 特殊字符：避免在值中使用
#    `$`、`''` 等特殊字符
# 3. 数字范围：人数和分值限制在 0-99
# 4. 日期格式：需使用 `YYYY.MM.DD` 格式
# 5. 模板文件：确保模板文件在正确目录下
#    (默认为 `./Template_模板` )

## 六、技术支持
# 遇到问题时检查：
#   1. 数据文件格式是否正确
#   2. 模板文件是否存在
#   3. 所有必填字段是否已填写
#   4. 字段名称是否拼写正确

# 通过本工具，您可以快速生成规范的学院文档，
